Load Balancer using http-proxy-middleware (Round-Robin)
This project demonstrates a basic Round-Robin Load Balancer implemented in Node.js using the Express framework and the http-proxy-middleware package.

The load balancer distributes incoming HTTP requests across multiple backend Express servers sequentially.
